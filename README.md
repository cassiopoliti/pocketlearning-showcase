# Pocket Learning

**Video production delivery management system.**

Pocket Learning is a web application built to organize and track the end-to-end workflow of video production projects — from assignment to final delivery. It was designed to manage the operational complexity of producing educational and corporate video content at scale.

🔗 **Product:** [pocketlearning.app](https://pocketlearning.app)

---

## What It Does

- **Project tracking:** manages the full lifecycle of video production deliveries.
- **Team coordination:** assigns tasks, tracks progress, and centralizes communication across production teams.
- **Delivery management:** controls deadlines, review cycles, and final delivery status for each video project.
